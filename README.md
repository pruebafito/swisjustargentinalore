# 🌿 Swiss Just Argentina — Bot de Lore

Bot de ventas y bienestar natural para **Lore Liffourrena**, Consultora oficial de Swiss Just Argentina.

## 📁 Estructura

```
swissjust-lore/
├── index.html   → Estructura HTML
├── styles.css   → Diseño visual
├── catalog.js   → 60 productos Swiss Just Argentina
├── app.js       → Lógica, IA, eventos y admin
└── README.md
```

## ✨ Funcionalidades

- 🤖 **Asistente Noa** — IA en español rioplatense con 60 productos
- 🛍️ **¡Vamos de compra!** — acceso directo a la tienda de Lore
- 💬 **WhatsApp** — contacto directo con Lore al tocar "Ver más" en productos
- 🎉 **Sistema de eventos** con sorteo de regalo presente
- ⚙️ **Panel de administración** exclusivo para Lore

## 🚀 Publicar en GitHub Pages

1. Subir los 4 archivos al repositorio de GitHub
2. Ir a **Settings → Pages → Deploy from branch → main → / (root)**
3. En 1-2 minutos disponible en `https://TU-USUARIO.github.io/REPO`

## ⚙️ Panel de administración

- Acceso: tocar **"⚙"** al pie de la página
